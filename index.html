Sumo Battlebot

Autonomous Sumo robot with ESP32, real-time sensor fusion and competition-grade ring control.

Autonomous sumo robot built on ESP32 and PlatformIO, updated with real competition logic. The firmware runs a 1 kHz control loop, reads four E18 photoelectric sensors to locate the opponent in front and at +/-45 degrees, and uses two front QRE1113 sensors with RC timing reads to detect the edge of the dohyo. The STANDBY, READY, VERSUS, and KILL state machine handles the official 5 s start delay, safety switch, rearming, and diagnostic telemetry.

The strategy prioritizes edge escape, spin recovery, short-term detection memory, alternating search, and direct or corrected attacks depending on the sensor mask. The project includes separate PlatformIO environments for the main firmware, wiring tests, and QRE serial tests, plus Python scripts for motor calibration and UDP control during trials.

Autonomous Driving with Neural Networks

Scaled vehicle making real-time semantic segmentation decisions for lane keeping and obstacle avoidance.

Autonomous driving platform for a scaled vehicle based on neural semantic segmentation. The system interprets each frame to separate drivable space from obstacles, estimates where the vehicle should move next, and turns the model output into steering and speed decisions.

The goal is not just to make the car move, but to build a complete architecture: video capture, neural inference, mask post-processing, autonomous decision-making, hardware command delivery, and safety mechanisms for switching between manual and automatic modes during tests.

Inverted Pendulum

Closed-loop PID balancing demo with real-time feedback on ESP32.

Real-time dynamic stabilization of an inherently unstable mechanical system. Deterministic implementation of a closed-loop PID controller running on a strict 100 Hz cycle on an ESP32 microcontroller. The system integrates digital filtering algorithms to reduce noise in sensor telemetry, continuously computes tilt error, and empirically tunes the proportional, integral, and derivative constants (Kp, Ki, Kd). The algorithm output generates high-resolution PWM signals for the motor drivers, enabling smooth, precise position correction without parasitic oscillations.

CV Sumo Upgrade

Next-gen vision system for opponent tracking and predictive maneuvering.

Evolution of the Sumo combat platform toward advanced spatial perception. This research-stage project aims to replace the one-dimensional infrared sensor array with a computer vision system integrated into the chassis. The architecture will offload visual processing to dedicated hardware running lightweight object-detection models (OpenCV) to identify the opponent's position, distance, and trajectory. The extracted vector data will feed the main ESP32 so it can run predictive navigation algorithms and dynamic interception tactics on the dohyo.
